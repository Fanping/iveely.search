iveely
==========


Iveely Search Engine 是由纯Java实现，依靠Iveely.Framework、Iveely.Computing、Iveely Database、Iveely.NLP 实现的分布式搜索（知识）引擎。<br/>
当前最新版本：0.8.0 。<br/>
